# 🛍️ АГЕНТ МОНИТОРИНГА ЦЕН И АКЦИЙ v2.0

**Автоматический агент с историей цен и умными рекомендациями о покупке!**

Ежедневно отслеживает цены на ваши товары в 8 магазинах и отправляет:
- ✅ **Краткое текстовое резюме** с рекомендациями (перед таблицей в email)
- ✅ **Срочные покупки** (товары на минимальной цене за месяц)
- ✅ **Товары с упавшей ценой** (сравнение с историей)
- ✅ **Рекомендации о покупке** (выгодный момент для покупки или ждать)
- ✅ **Подробный HTML отчёт** с полной информацией

## ✨ Новое в v2.0

### 🗄️ Историческая база данных цен
- Агент автоматически сохраняет все цены в `price_history.db`
- Хранит данные за 30+ дней
- Позволяет видеть тренды и минимумы цен

### 📊 Сравнение с историей
- **Срочные покупки**: товары на минимальной цене за последний месяц
- **Падение цены**: видишь, когда цена снизилась (с рекомендацией купить)

### 💡 Умные рекомендации
Агент сам рекомендует:
- ✅ **Купи СЕЙЧАС** — когда цена на минимуме
- 📉 **Выгодно** — когда цена упала по сравнению с историей
- ⏸️ **Подожди** — когда цена повышается (лучше покупать потом)
- 💰 **Переключись на другой магазин** — когда разница > 50₽

### 📱 Текстовое резюме в начале письма
- Не надо читать всю таблицу
- Главные рекомендации в начале email
- Быстрый обзор: что купить, где и почём

- **Python 3.8+** ([скачать](https://www.python.org/downloads/))
- **Email аккаунт Gmail** (рекомендуется)
- **Интернет подключение**

## 🚀 Установка

### Шаг 1: Создать папку проекта

```bash
# Создаём папку для проекта
mkdir price_monitor
cd price_monitor
```

### Шаг 2: Скопировать файлы

Скопируй эти файлы в папку проекта:
- `price_monitor_agent.py` — основной агент
- `scheduler.py` — планировщик (опционально)
- `grocery_analysis.xlsx` — твой файл с товарами

### Шаг 3: Установить зависимости

```bash
# Установить Python пакеты
pip install pandas openpyxl schedule requests
```

### Шаг 4: Настроить email (ВАЖНО!)

#### 4.1. Если у вас Gmail:

1. **Включить двухфакторную аутентификацию** (если ещё не включена):
   - Перейди на [myaccount.google.com](https://myaccount.google.com)
   - Security → 2-Step Verification

2. **Создать "App Password"**:
   - [Перейди сюда](https://myaccount.google.com/apppasswords)
   - Выбери "Mail" и "Windows Computer"
   - Google выдаст 16-символьный пароль
   - **Скопируй этот пароль** — он понадобится

3. **Установить переменную окружения**:

**Windows (PowerShell):**
```powershell
$env:EMAIL_APP_PASSWORD = "твой_16_символьный_пароль"
python price_monitor_agent.py
```

**Windows (Command Prompt):**
```cmd
set EMAIL_APP_PASSWORD=твой_16_символьный_пароль
python price_monitor_agent.py
```

**Mac/Linux (Bash):**
```bash
export EMAIL_APP_PASSWORD="твой_16_символьный_пароль"
python price_monitor_agent.py
```

#### 4.2. Если используешь другой email (Yandex, Mail.ru и т.д.):

Отредактируй `price_monitor_agent.py`, измени эти строки:

```python
EMAIL_CONFIG = {
    'sender': 'твой_email@mail.com',
    'app_password': os.getenv('EMAIL_APP_PASSWORD', ''),
    'recipient': 'ivanaryzhkov@icloud.com',
    'smtp_server': 'smtp.mail.com',  # Поменяй на правильный SMTP
    'smtp_port': 587
}
```

SMTP серверы:
- **Gmail**: `smtp.gmail.com` (587)
- **Yandex**: `smtp.yandex.ru` (587)
- **Mail.ru**: `smtp.mail.ru` (587)
- **Outlook**: `smtp-mail.outlook.com` (587)

## ⚙️ Конфигурация

### Изменить список товаров

Отредактируй файл `grocery_analysis.xlsx`:
- Лист: `🛒 Все товары`
- Добавь/удали товары в столбце "Товар"
- **Сохрани файл**

Агент автоматически загрузит новый список при следующем запуске.

### Изменить магазины

В файле `price_monitor_agent.py` найди раздел `STORES`:

```python
STORES = {
    'perekrestok': {'name': 'Перекрёсток', ...},
    'pyaterochka': {'name': 'Пятёрочка', ...},
    # ... остальные магазины
}
```

Чтобы удалить магазин из мониторинга, закомментируй его строку:

```python
# 'perekrestok': {'name': 'Перекрёсток', ...},
```

### Изменить время запуска

В файле `scheduler.py` найди строку:

```python
schedule.every().day.at("07:00").do(run_price_monitor)
```

Измени `"07:00"` на нужное время (например, `"08:30"`).

### Изменить email получателя

В файле `price_monitor_agent.py` измени строку:

```python
'recipient': 'ivanaryzhkov@icloud.com',  # ← Поменяй на свой email
```

## 🏃 Запуск

### Вариант 1: Один раз (для теста)

```bash
python price_monitor_agent.py
```

Агент:
- ✅ Загрузит товары из Excel
- ✅ Проверит цены в магазинах
- ✅ Сгенерирует отчёт
- ✅ Отправит на почту
- ✅ Сохранит копию в папке `reports/`

### Вариант 2: Автоматически каждый день в 7:00

```bash
python scheduler.py
```

Окно останется открытым. Агент будет запускаться каждый день в 7:00.

**Чтобы остановить**, нажми `Ctrl+C`.

### Вариант 3: Фоновый процесс (Windows)

Создай файл `run_scheduler.bat`:

```batch
@echo off
python scheduler.py
pause
```

Запусти двойным кликом. Окно будет работать в фоне.

### Вариант 4: Фоновый процесс (Mac/Linux)

```bash
nohup python scheduler.py > scheduler.log 2>&1 &
```

Проверить статус:
```bash
ps aux | grep scheduler.py
```

Остановить:
```bash
pkill -f scheduler.py
```

### Вариант 5: Запуск по расписанию через Windows Task Scheduler

1. Открой **Task Scheduler** (`taskschd.msc`)
2. "Create Basic Task"
3. Имя: "Price Monitor"
4. Триггер: "Daily" в 07:00
5. Action:
   - Program: `C:\Python312\python.exe` (путь к Python)
   - Arguments: `C:\path\to\price_monitor_agent.py`
   - Start in: `C:\path\to\project\`

## 📝 Управление товарами

### Как добавить товар

1. Открой `grocery_analysis.xlsx`
2. Перейди на лист `🛒 Все товары`
3. Добавь новую строку с названием товара
4. **Сохрани файл**

Примеры названий товаров (важны точные названия!):
- Молоко Простоквашино 1.4л
- Яйцо С1/С2 10шт
- Масло оливковое 500мл
- Кофе Monarch 800г
- Сыр Ламбер 230г

### Как удалить товар

1. Открой `grocery_analysis.xlsx`
2. Удали строку с товаром
3. **Сохрани файл**

### Как приоритизировать товары

Чтобы агент отслеживал только самые важные товары (для скорости):

В файле `price_monitor_agent.py` измени строку:

```python
for product in self.products[:10]:  # ← Измени 10 на нужное число
```

Например:
- `:5` — мониторить только топ-5 товаров
- `:20` — мониторить топ-20 товаров

## 🔧 Помощь и вопросы

### Проблема: "EMAIL_APP_PASSWORD не установлен"

**Решение:**
- Убедись, что установил переменную окружения перед запуском
- Для Google: создай App Password ([инструкция](#шаг-4-настроить-email))
- Проверь, что пароль скопирован правильно (без пробелов)

### Проблема: "Файл grocery_analysis.xlsx не найден"

**Решение:**
- Убедись, что файл находится в той же папке, что и `price_monitor_agent.py`
- Проверь название файла (чувствителен к регистру на Mac/Linux)

### Проблема: "Отчёт не приходит на почту"

**Решение:**
1. Проверь файл `price_monitor.log` на ошибки
2. Убедись, что электронная почта отправителя правильная в `EMAIL_CONFIG`
3. Проверь, что в почте включены "менее безопасные приложения" (если используешь не App Password)
4. Проверь папку "Спам" в почте

### Проблема: "ModuleNotFoundError: No module named 'pandas'"

**Решение:**
```bash
pip install pandas openpyxl schedule requests
```

### Проблема: Планировщик не запускается в точное время

**Решение:**
- Компьютер должен быть включён в нужное время
- Для гарантированного выполнения используй облачные сервисы (см. ниже)

## ☁️ Продвинутый уровень: Облачный запуск

Если хочешь, чтобы агент работал в облаке (без необходимости держать компьютер включённым):

### Вариант 1: GitHub Actions (Бесплатно)

1. Создай репозиторий на GitHub
2. Положи файлы в репозиторий
3. Создай файл `.github/workflows/price-monitor.yml`:

```yaml
name: Price Monitor

on:
  schedule:
    - cron: '0 4 * * *'  # 7:00 по московскому времени

jobs:
  monitor:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: 3.9
      - name: Install dependencies
        run: pip install pandas openpyxl requests
      - name: Run price monitor
        env:
          EMAIL_APP_PASSWORD: ${{ secrets.EMAIL_APP_PASSWORD }}
        run: python price_monitor_agent.py
```

4. Добавь секрет `EMAIL_APP_PASSWORD` в Settings → Secrets

### Вариант 2: Railway.app (Платная, но просто)

1. Развернуть на Railway: https://railway.app
2. Настроить переменные окружения
3. Настроить cron job для ежедневного запуска
